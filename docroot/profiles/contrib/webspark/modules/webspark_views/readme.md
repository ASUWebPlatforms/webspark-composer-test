## INTRODUCTION

This module provides a default card view format. This allows view results to be displayed in a card arrangement as default cards.

## CONFIGURATION

This module is configured within the views UI. Create a view of content, then select the Default Card View format to configure. You will need to map your content's fields to the parts of a card.
